The project is a web application for a GYM. The main purpose is for users to be able authenticate and with their authorization they can have specific actions. 
User -> To book specific trainigs and take meassurements of their progress (Weight, milerun, strength) and see their change in statistics with visual representation.
Trainer -> To create new trainnings, manage existing trainigs, manage rooms, and access user information, create new users.

 2+ team-defined stories -> 
 - As a user I can track my stats(Weight, height, strenght, resistence)  change over time.
 - User receibes a whatsapp message regarding their class.
 - Trainer can make a room unabailable, and it cant be booked and current bookings get different status.
