Aplicación para definir rutinas de ejercicio. 

Both ->
Log in / out
Register (Users, trainers via invite) 3 role admin to create new account for trainers?

Trainer -> 
Curate the catalog. As an admin, I can add, edit, and remove catalog items.
Crear entrenamientos, editar entrenamientos, borrar entrenamientos
Read a report
Entrenamientos más populares, top usuarios

Usuario -> 
+Extra -> Checar sus estadísticas sobre tiempo (Peso, resistencia, fuerza) en graficas
+Extra -> Ver calendario con actividades ya pasadas
+Extra -> push notification 
+Extra -> Logros & Racha
Browse & search the catalog. As a consumer, I can list the catalog and narrow it (search or filter).
Puede elegir entrenamientos.
Filtrar por tipo de entrenamiento, dificultad
Busqueda
See my own — and only my own. As a consumer, I can view my transaction history, and I can never see anyone else's
Historial de entrenamientos.
